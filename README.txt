# Hospital Information System - Executable

## How to Run
1. Ensure MySQL server is running on `192.168.1.88` with database `his_db`.
2. Double-click `main.exe` to start the application.
3. Default login: admin / admin123

## Requirements
- MySQL server accessible at 192.168.1.88
- Database `his_db` with tables created (run the app once to create tables)

## Notes
- This is a standalone executable created with PyInstaller.
- No Python installation required on the target machine.
- MySQL connector is bundled.
